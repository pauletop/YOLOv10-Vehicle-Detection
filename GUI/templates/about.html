About

Vehicle detection is vital in computer vision, especially for autonomous driving, traffic monitoring, and surveillance. While segmentation is more accurate, it requires more resources. YOLO (You Only Look Once) is a real-time object detection algorithm that balances speed and accuracy.
This system uses the latest YOLOv10 model (as of June 25, 2024) to detect vehicles efficiently. The model is trained on a hand-labeled COCO dataset with five classes: car, van, truck, bus, and motorcycle. Show more ...

This GUI allows users to upload images or videos to detect vehicles, providing a user-friendly interface for the detection process. It's a part of the final project for Computer Vision course at Ton Duc Thang University. Show less ...
